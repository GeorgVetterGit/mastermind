Mastermind

Project Report

Introduction

Mastermind is a digital recreation of the classic logic game, developed in Python using the Pygame framework. The objective of the game is to guess a hidden colour code within a limited number of attempts. After each guess, the player receives feedback in the form of black and white pegs, indicating how close the guess is to the secret code.

Game Overview

The game follows the classic Mastermind rules:

  • A secret code consisting of coloured pegs is randomly generated.
  • The player has a fixed number of attempts to crack the code.
  • After each guess, the game provides feedback:
    • Black peg: Correct colour in the correct position.
    • White peg: Correct colour in the wrong position.
  • The game ends when the code is cracked or all attempts are used up.

Features

  • Intuitive graphical interface built with Pygame
  • Colour selection via mouse click
  • Visual feedback after each guess
  • Adjustable difficulty (number of colours and code positions)
  • Playable in the browser thanks to Pygbag compilation

Technical Implementation

The project was implemented entirely in Python and uses Pygame for graphical rendering. The game logic is cleanly separated from the rendering layer, enabling easy extensibility.

  • Language: Python 3.12
  • Framework: Pygame
  • Web Deployment: Pygbag (Python-to-WebAssembly compilation)
  • Architecture: Modular design with separated game logic and UI

Game Mechanics

The core mechanic is based on deductive reasoning. By combining the clues from previous attempts, the player can systematically narrow down the possible solutions. The feedback algorithm compares the guess against the secret code and calculates the number of black and white pegs according to the official Mastermind rules.

Conclusion

This project demonstrates the implementation of a classic board game as an interactive web application. By using Pygame and Pygbag, the game is playable both locally and in the browser. The project serves as an example of small-scale game development with Python and shows how traditional game concepts can be brought into the digital realm.
